#  OCR Box

一个轻量级、高度可定制的浏览器截图 OCR 扩展，基于免费的 [ocr.space API](https://ocr.space/OCRAPI)。

![[此处可以放一张 GIF 动图，展示从截图到弹出结果框的全过程]](https://via.placeholder.com/800x400.png?text=OCR+Box+功能演示)

---

## ✨ 功能特性

* **⚡️ 快速截图**：按下快捷键或点击图标，即可像微信截图一样（四周暗化）选取屏幕区域。
* **🔌 API 驱动**：使用 `ocr.space` API，识别率高，且完全免费（有每日额度限制）。
* **🔑 用户配置**：
    * **API Key**：用户**必须**提供自己的 API Key，确保了隐私安全，也使用了用户自己的免费额度。
    * **自定义快捷键**：支持自定义 `Alt`, `Ctrl`, `Shift` + 任意主键的组合。
    * **引擎切换**：支持“引擎1”（速度优先）、“引擎2”（中日韩优化）以及**“自动”**模式。
* **🤖 智能备援**：
    * 在“自动”模式下，优先尝试引擎2。
    * 如果引擎2失败、超时（10秒）或返回服务器内部错误（E208），将**自动无缝切换到引擎1**重试。
* **🎨 现代化结果框**：
    * **下一张**：在结果框上直接开始下一次截图。
    * **复制**：一键复制所有识别文本。
    * **浅/深主题**：自适应的浅色/深色主题切换。
    * **设置**：快速跳转到选项页。
    * 点击标题栏或按 `ESC` 键可快速关闭。
* **💡 智能提示**：
    * 在 API Key 未设置时，会弹出提示并提供**可点击的“立即设置”链接**。
    * 在 API Key 错误或服务器错误时，会弹出友好的中文提示。

## 📦 安装

### 1. (推荐) 从 Edge 扩展商店安装

1.  前往 Microsoft Edge 扩展商店。
2.  搜索 "OCR Box" (或您上架时使用的名称)。
3.  点击“获取”按钮。

*(上架成功后请替换为商店链接)*

### 2. (用于开发) 本地加载

1.  克隆 (Clone) 或下载本仓库的 `.zip` 文件并解压。
2.  打开 Edge 浏览器，在地址栏输入 `edge://extensions/` 并回车。
3.  开启页面左下角的“**开发人员模式**”。
4.  点击“**加载已解压缩的扩展**”，选择你刚刚解压的文件夹（即包含 `manifest.json` 文件的那个文件夹）。

---

## 🚀 首次设置 (必读)

为了保护您的隐私并使用您自己的免费额度，本插件**需要您手动配置** `ocr.space` 的 API Key。

1.  在 Edge 浏览器右上角找到本插件图标，**右键点击**，选择“**选项**”。
2.  在打开的选项页面中，您会看到获取 API Key 的教程。
3.  点击 [ocr.space 免费 API 页面](https://ocr.space/OCRAPI#free) 链接，注册一个 **FREE Plan** 账户。
4.  登录后，在您的账户信息中找到并复制您的 API 密钥（API Key）。
5.  将密钥粘贴到选项页的“API Key”输入框中（输入框为密码模式，可点击“显示”来检查）。
6.  点击“**保存设置**”。

您现在可以开始使用了！

---

## 🖱️ 如何使用

1.  **方式一 (快捷键)**：在任意网页上，按下您在选项页设置的快捷键 (默认为 `Alt` + `0`)。
2.  **方式二 (图标)**：点击浏览器工具栏右上角的 OCR Box 插件图标。

触发后，屏幕会变暗，拖动鼠标选择您想识别的区域即可。识别结果将自动在右下角弹出。
